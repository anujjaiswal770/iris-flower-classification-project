# Iris Flower Classification

## Project Overview

This project analyzes the famous Iris dataset and uses machine learning techniques to identify patterns among different iris flower species based on sepal and petal measurements.

## Dataset

* Dataset: Iris Dataset
* Records: 150 flowers
* Features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* Classes:

  * Setosa
  * Versicolor
  * Virginica

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Data Exploration
2. Data Visualization
3. Species Encoding
4. K-Means Clustering
5. Cluster Evaluation
6. Confusion Matrix Analysis

## Key Analysis

* Visualized relationships between sepal and petal measurements.
* Created 2D and 3D scatter plots.
* Applied the Elbow Method to determine the optimal number of clusters.
* Grouped flowers into clusters using K-Means.

## Skills Demonstrated

* Data Visualization
* Exploratory Data Analysis
* Feature Analysis
* Clustering Techniques
* K-Means Algorithm
* Model Evaluation

## Conclusion

The project demonstrates how unsupervised machine learning can be used to identify natural groupings within the Iris dataset. K-Means clustering successfully separated the flower species based on petal and sepal characteristics.
